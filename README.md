# ShopX 🛍️

A full-featured e-commerce frontend built with React.

## Live Demo
🔗 [https://shopx-bijy.vercel.app](https://shopx-bijy.vercel.app)

## Features
- 🛒 Add to cart with quantity selector
- 🔍 Search and filter by category
- 🌙 Dark/Light mode
- 💾 Cart persists with localStorage
- 📱 Fully responsive design

## Tech Stack
- React + Vite
- Context API
- React Router
- CSS Modules
- FakeStore API
- Deployed on Vercel
